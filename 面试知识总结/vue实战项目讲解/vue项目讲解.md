#vue项目讲解

##前后端分离的开发方式

###前端项目技术栈
- Vue
- Vue-router
- Element-UI
- Axios
- Echarts

###后端项目技术栈
- Node.js
- Express
- Jwt （状态保存的一个工具，实现登录记录功能，即sessionId）
- Mysql
- Sequelize（操作数据库的框架）

###前端项目初始化步骤
- 安装Vue脚手架
- 通过Vue脚手架创建项目 - （通过黑屏终端 vue ui 命令可视化创建vue项目）
- 配置Vue路由
- 配置Element-UI组件库 - （通过vue可视化安装插件搜索：vue-cli-plugin-element）
- 配置Axios库  -   （通过vue可视化安装依赖（运行依赖）搜索：axios）
- 初始化git远程仓库
- 将本地项目托管到github中或码云中
